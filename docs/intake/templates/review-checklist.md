# Review checklist

Use before promoting anything from `docs/intake/` to `public/`.

- [ ] Correct intake folder and descriptive filename.
- [ ] Status assigned from `REVIEW_STATUS.md`.
- [ ] No private IDs, addresses, credentials, API keys, financial data or private network details.
- [ ] Caleb has permission to publish it.
- [ ] Asset strengthens a specific portfolio section or case study.
- [ ] Optimized for web size and quality.
- [ ] Accessible alt/caption context is known if used visually.
- [ ] Public copy placed in the correct `public/` destination.
- [ ] `MANIFEST.md` updated.

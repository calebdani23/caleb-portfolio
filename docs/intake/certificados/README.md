# Certificados intake

This folder stores source certificate PDFs grouped by program. Public-safe copies and preview images live under `public/docs/certificados/` and `public/images/certificates/`.

Default status for new uploads: `needs-review`.

## Current folders

- `programa-google-it-support/` — Google IT Support Professional Certificate and course certificates. Original source label: `Programa Google IT Support/`.
- `programa-oracle-alura/` — Oracle/Alura source certificates. Original source label contained a typo: `Programa Orcale + Alura/`; normalized path fixes it to Oracle.

## Rules

- Keep source PDFs here even when public copies exist.
- Check every PDF for private IDs, account data and unnecessary personal details before promotion.
- Track promoted public PDFs in `../MANIFEST.md`.

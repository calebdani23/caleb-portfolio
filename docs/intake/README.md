# Intake de contenido del portfolio

Esta carpeta es un espacio de staging para subir material fuente antes de decidir qué assets pasan a `public/` y se muestran en el sitio.

## Cómo usarla

1. Copia `PORTFOLIO_CONTENT_TEMPLATE.md` y rellénala con datos reales.
2. Sube screenshots, mockups, diagramas, videos/gifs y notas en la carpeta del proyecto correspondiente.
3. Guarda certificados, reconocimientos y variantes de foto en sus carpetas dedicadas.
4. Usa `shared/` para assets reutilizables entre varios proyectos.
5. No optimices ni renombres de forma definitiva todavía: esta carpeta es para material fuente y selección editorial.

## Convención sugerida de nombres

Usa nombres descriptivos y simples:

```text
zalmar-travel-booking-flow-v1.png
visual-runner-server-dashboard-2026-05.png
profile-portrait-dark-bg.svg
certificado-fastapi-curso.pdf
```

Cuando un asset esté aprobado para producción, se puede copiar/optimizar hacia `public/` y referenciar desde los datos del sitio.
